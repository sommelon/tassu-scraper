# tassu-scraper (Outdated)

An [epc.lib.tuke.sk](https://epc.lib.tuke.sk/PrehladPubl.aspx) scraper, built using Java and the Selenium framework.
The scraped data is transformed and loaded into a database.
The final task was to design a data warehouse and populate it with data aggregated from other teams (the data was provided as an SQL script).
